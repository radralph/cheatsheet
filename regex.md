##REGULAR EXPRESSION

###Anchors
```
^  - Start of line
\A - Start of string
$  - End of line
\Z - End of string
\b - Word boundary
\B - Not word boundary
\< - Start of word
\> - End of word
```

###Character Classes
```
\c - Control character
\s - White space
\S - Not white space
\d - Digit
\D - Not digit
\w - Word
\W - Not word
\x - Hexadecimal digit
\O - Octal Digit
```
