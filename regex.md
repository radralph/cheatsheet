##REGULAR EXPRESSION

###Anchors
```
^  - Start of line
\A - Start of string
$  - End of line
\Z - End of string
\b - Word boundary
\B - Not word boundary
\< - Start of word
\> - End of word
```

###Character Classes
```
\c - Control character
\s - White space
\S - Not white space
\d - Digit
\D - Not digit
\w - Word
\W - Not word
\x - Hexadecimal digit
\O - Octal Digit
```

###Quantifiers
```
*  - 0 or more {3} Exactly 3
+  - 1 or more {3,} 3 or more
?  - 0 or 1 {3,5} 3,4 or 5
```
###Escape Sequences
```
\  - Escape following characters
\Q - Begin literal sequence
\E - End literal sequence
```
